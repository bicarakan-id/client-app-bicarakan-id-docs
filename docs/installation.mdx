## Requirements
- [Flutter v.2.8.0 =< v.3.0.0][web flutter]
- Git
- Internet Connection

## Platform Support
- Android
- iOS

## Steps
- clone repository
```shel
  git clone https://github.com/bicarakan-id/client-app-bicarakan-id.git
```
- Open with your text editor / IDE
- run flutter pub get
```shel
  flutter pub get
```
- run project with flavor
```shel
  flutter run --debug --flavor dev
```






[web flutter]: https://flutter.dev
